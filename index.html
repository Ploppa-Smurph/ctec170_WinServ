CTEC-170 Windows Server wk01 Intro

Students will be involved with the installation, storage, and compute functionalities available in Windows Server 2019. Students perform general installation tasks, 
including installing and configuring a Windows server, implementing Hyper-V and rapid service deployment. Students will gain experience with local and server storage 
solutions including managing data storage, configuring and managing network services including remote access services. Students will become familiar configuring web 
services and cloud technologies and securing a Windows network. Finally, students will be introduced to active directory and account management.
